# WebScraper

Este mini projeto é de um raspador de dados feito em Python utilizando a biblioteca BeautifulSoup, para a matéria Linguagem de Programação II.
O objetivo é coletar manchetes de notícias de diferentes jornais e exibi-las no terminal, mas inicialmente começarei com anuncio de 1 jornal, buscando as 5 principais notícias do dia e incrementar outros jornais no decorrer do projeto.
É um mini projeto simples, para iniciantes, que mostra como automatizar a coleta de informações da web.
Será utilizado Python 3, Requests (para acessar páginas da web) e a biblioteca BeautifulSoup4 (pypi.org in Bing) para interpretar o HTML.
